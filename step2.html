Your First Java Project

  1. Launch Eclipse

    Open Eclipse, you will be prompted to set the directory where all of your projects will be saved.

  2. Create a New Project

    After opening Eclipse you will be greeted by a welcome window

    To create a new project navigate to:

    File --> New --> Java Project

  3. Your Project

    After choosing New Project you will be prompted to give your project a name

    Enter "HelloWorld" in the Project Name box and click finish

  4. Time to Create a Class File

    Head over to the next page to create your class file.
