Let's Get Started!

  1. Get an IDE

    In order to begin developing Java you fist need to install an IDE.

    An integrated development environment (IDE) is a software application that provides comprehensive facilities to computer programmers for software development.

    An IDE is important because it compiles the developers code and helps to give suggestions on how to fix errors. It also helps the user with source control, saving, sharing, and exporting their projects.

  2. Download Java

    In order to start developing for Java you must make sure you have Java installed on your machine.

    Follow the link to check if Java is installed: https://www.java.com/verify/

  3. Eclipse, An open source IDE

    While there are many different IDEs a developer to choose from, one of the most popular is called Eclipse.

    To get started click here to download Eclipse.

  4. Setting up you're environment

    1. Extract the files you have downloaded and run the Eclipse Installer.

    2. After opening the installer, choose the first option, "Eclipse IDE for Java Developers".

    3. Follow the rest of the prompts, choose your installation directory and wait for the installer to complete

    4. After the installation is complete choose the "Launch" option to start eclipse
