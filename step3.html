Creating a Java Class

  1. Create Your Class

    Under the Package Explorer find your project and right click

    Navigate to: New --> Class

  2. Naming Your Class

    A Java Class file sits inside of your porject, you must give your class file a name.

    Enter "main" into the Name and click finish.
