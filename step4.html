Time to Write Some Code

  1. Main Method

    In order to run any Java program the computer must be told where to begin running the code.

    This is called the main method, any code you put inside the main method is immediately run when the program is called

    Inside the class brackets, Enter the following string of text:

    public static void main(String[] args){ }

  2. Print Hello World

    Now, inside of the brackets in the main method type:

    System.out.println("Hello World");

  3. Run Your Program

    Now that you have entered all the code needed, it's time to run the program.

    To run the program, find the green play button at the top of Eclipse's navigation bar.

    If everything goes well, you should see "Hello World" appear under the console tab.

Congratulations

You have now written your very first Java Program!
